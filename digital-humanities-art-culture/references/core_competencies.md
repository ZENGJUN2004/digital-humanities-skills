# 数字人文核心技能与能力体系

## 1. 技术技能

### 1.1 数字图像处理
- **核心能力**：艺术作品的形式分析、风格识别、图像增强与修复
- **应用领域**：艺术史研究、文化遗产保护、艺术教育
- **关键技术**：图像增强、特征提取、风格分析、风格转换、图像分割
- **工具与软件**：OpenCV、scikit-image、Pillow、Adobe Photoshop、GIMP

### 1.2 3D建模与扫描
- **核心能力**：文物数字化、文化遗产三维记录与重建
- **应用领域**：文化遗产保护、虚拟展览、考古研究
- **关键技术**：激光扫描、摄影测量、3D建模、纹理映射、渲染
- **工具与软件**：MeshLab、Blender、Agisoft Metashape、ContextCapture

### 1.3 编程与软件开发
- **核心能力**：创建自定义工具、开发数字人文平台
- **应用领域**：数字人文研究、教育、文化传播
- **关键技术**：Python、JavaScript、HTML/CSS、数据结构、算法设计
- **工具与软件**：Visual Studio Code、Jupyter Notebooks、GitHub

### 1.4 数据分析与可视化
- **核心能力**：文化数据的定量研究、模式识别
- **应用领域**：文化研究、艺术市场分析、文化政策评估
- **关键技术**：统计分析、网络分析、数据可视化、机器学习
- **工具与软件**：Pandas、scikit-learn、NetworkX、Tableau、Power BI、Plotly

### 1.5 文本分析与编码
- **核心能力**：文本挖掘、数字编辑、文本编码
- **应用领域**：文学研究、历史研究、语言学研究
- **关键技术**：自然语言处理、文本挖掘、TEI编码、语义分析
- **工具与软件**：NLTK、spaCy、Voyant Tools、 Oxygen XML Editor

### 1.6 数字存档与保存
- **核心能力**：数字文化遗产的长期保存、格式转换
- **应用领域**：图书馆学、档案学、文化遗产保护
- **关键技术**：数字保存策略、格式标准化、元数据管理、备份策略
- **工具与软件**：DROID、PRONOM、Archivematica
